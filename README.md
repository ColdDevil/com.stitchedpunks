# StitchedPunks.com

- This repository contains the source code of the `StitchedPunksShop` and the `StitchedPunksNFT` smart contracts for [StitchedPunks.com](https://stitchedpunks.com)
- The official shop contract is deployed at the address `0x9f4263370872b44EF46477DC9Bc67ca938e129c6` and can be seen here: https://etherscan.io/address/0x9f4263370872b44ef46477dc9bc67ca938e129c6
- The contract uses the `Ownable` and `Context` libraries from OpenZeppelin, which can be found here: https://github.com/OpenZeppelin/openzeppelin-contracts/
- The official NFT contract is deployed at the address `0xEE2F4a0572B1EFbaEca070f8F08d2917EAC38dc9` and can be seen here: https://etherscan.io/address/0xee2f4a0572b1efbaeca070f8f08d2917eac38dc9
